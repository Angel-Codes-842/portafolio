# Portafolio Web - Ángel Gómez

Portafolio personal con estética retro-tech 8-bit minimalista, construido con React y Tailwind CSS.

## 🚀 Demo

[Ver portafolio en vivo](#) *(agregar URL cuando esté desplegado)*

## 🛠️ Tecnologías

- React 18
- Tailwind CSS
- Vite
- React Icons

## 📦 Instalación

```bash
# Clonar repositorio
git clone https://github.com/Angel-Codes-842/portafolio-web.git

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build
```

## 📁 Estructura

```
src/
├── components/
│   ├── layout/      # Navbar, Footer
│   ├── sections/    # Hero, About, Projects, Contact
│   └── ui/          # Componentes reutilizables
├── data/            # Datos de proyectos
├── hooks/           # Custom hooks
└── index.css        # Estilos globales
```

## ✨ Características

- Diseño responsive (mobile-first)
- Animaciones retro (CRT fade-in, cursor parpadeante)
- Carrusel de imágenes en proyectos
- Lightbox para ver imágenes ampliadas
- Iconos de tecnologías
- Formulario de contacto
